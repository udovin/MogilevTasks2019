УСЛОВИЕ:
Сегодня у мальчика Бита юбилей - 5 лет! Конечно, Бит не мог не попросить у родителей очередную игрушку
на день рождения - линейка Рубика.

Линейка Рубика - это простая игрушка, состоящая из N шестиугольничков, сцепленных между собой. Каждый
шестиугольничек имеет в качестве допустимых значений числа от 0 до 5, по умолчанию все значения равны
нулю. Соответственно, если повернуть шестиугольничек, то его значение изменится на единицу большее, а
в случае числа 5 - станет равным 0.

Бит повершул сначала каждый второй кубик (соответственно 2, 4, 6...). Затем он повернул каждый третий
кубик (соответственно 3, 6, 9...). Далее - каждый четвертый. Он продолжал так до тех пор, пока не
повернул каждый N-ый кубик, тем самым повернув только кубик с номером N.

Биту стало интересно: сколько шестиугольничков сейчас показывают число 1?


ВХОДНЫЕ ДАННЫЕ:
Единственная строка входного файла содержит одно число N (1 ≤ N ≤ 10^18) – число шестиугольничков в
линейке Рубика.


ВЫХОДНЫЕ ДАННЫЕ:
Единственная строка выходного файла содержит одно целое число – количество шестиугольничков, показывающих 1.


ПРИМЕРЫ:
in: 100
out: 2

in:  100000000
out: 69


ПОЯСНЕНИЯ:
В первом примере - эо кубики с номерами 1 и 69.


ОГРАНИЧЕНИЯ:
1) N ≤ 10^7   (20 баллов)
2) N ≤ 10^8   (50 баллов)
3) N ≤ 10^14  (80 баллов)
4) N ≤ 10^18  (100 баллов)
